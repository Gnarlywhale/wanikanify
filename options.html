Public API Key

Your public API key can be found on your account page at WaniKani. Your API key is only used to access your vocabulary list.

Vocabulary Cache

Your studied vocabulary words are cached locally to prevent overloading the WaniKani servers. You may clear the cache manually by hitting this button. The cache will be repopulate automatically.

Run On

You may have WaniKanify run automatically after a page loads, or when the Crabigator is clicked.

SRS Filters

Wanikanify will only substitute words in the checked SRS groups.

Google Spreadsheet Import List

Similar to "Custom Vocabulary", but more extensive. Allows users to use shared vocabulary lists. Don't freak out, it's easy to use.

  1. Make sure the spreadsheet has been published. File->Publish to web.
  2. Each row tells WaniKanifier to import a single spreadsheet. NOTE: A spreadsheet collection can contain multiple sheets, see below.
  3. Key: The key uniquely identifies the spreadsheet collection (group of sheets). Grab the "key" from the URL, and put it below. Looks something like this: 1lIo2calXb_GtaQCKLr989-Ma_hxXlxFsHE0egko-D9k
  4. Make sure the columns in the spreadsheet are all named (has a header row at the top). Do not leave any empty.
  5. From column header: The name of the column (in the header) we want to map from (column of English words).
  6. To column header: The name of the column (in the header) we want to map to (column of Kanji/Japanese words).
  7. Delimiter: The English word column can contain multiple words, separated by a delimiter (default is a comma). Spaces after the delimiter are okay. But you probably don't want a delimiter of just whitespace.
  8. Removing an entry will also remove it from the cache.
  9. Sheet Name: The sheet name should only contain one sheet in the spreadsheet collection. The name is found at the bottom of the screen and it looks like a tab.
  10. If you use Google Chrome Sync, these settings will be synced, but THE VOCAB WILL NOT! You will have to reimport them (on the other computer just click "import"). The reason is because it's too much data to sync.
